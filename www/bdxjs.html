Comment ?

  • Un langage de programmation
    • strictement typé
    • moderne, souple (pas Java)
  • Un compilateur
    • transforme *.hx en source/binaire
    • extremement rapide
    • extensible /macros
  • Une bibliothèque standard
    • cross platform
    • spec + tests unitaires
  • Des API pour la platforme
    • facile à compléter/modifier

Exemple

Un exemple de code pour se faire une idée: 

function test() {
  var str = "Hello world";
  var chars = [for( i in 0...str.length ) str.charCodeAt(i)];
  var tot = 0;
  for( c in chars )
	tot += c;
  return tot;
}

Inférence

L'inférence de type en action: 

function test() {
  var str : String = "Hello world";
  var chars : Array<Int> = [for ...];
  var tot : Int = 0;
  var i = 0, len = chars.length;
  while( i < len )
	tot += chars[i++];
  return tot; // Int
}

Macros

Un programme qui s'éxecute à la compilation. 

		macro function getFile( file : String ) {
			var content = sys.io.File.getContent(file);
			return macro $v{content};
		}
		//...
		var str = getFile("config.json");

Génere dans le .js: 

		var str = '{ "user" : "warp", "pass" : "root" }';

Macros (2)

Encore un peu mieux: 

		macro function getFile( file : String ) {
			var content = sys.io.File.getContent(file);
			var obj = haxe.Json.parse(content);
			return macro $v{obj};
		}
		//...
		var o = getFile("config.json");
		o.|

Génere dans le .js: 

		var str = { user : "warp", pass : "root" };

Macros (3)

Haxe Records ORM: 

		var u = User.manager.search($age >= 18 && $name.like("X%"));

Génere dans le code: 

		var u = User.manager.query("age >= 18 AND name LIKE 'X%'");

  • vérifie que l'on ne fait pas n'importe quoi
